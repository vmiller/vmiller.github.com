MDT Database Scripts

May 10th, 2013

I recently dusted off a Powershell script that I wrote to do a batch upload of computer records to a MDT database. It builds on Powershell modules provided by Michael Neihaus. My original LoadComputers.ps1 script reads in a csv file and does some basic data validation before creating the new records.

My reason for dusting off this script was to delete a batch of computers. I modified the logic and that became the DeleteComputers.ps1 script.

These scripts can most certainly use some more polish, and will need modified to fit the particular environment they are being used in. I hope they might give someone else a good starting point.
